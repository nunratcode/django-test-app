# Django App с древовидным меню

Это Django-приложение реализует **древовидное меню**, которое хранится в базе данных, редактируется через стандартную админку и отображается на сайте через кастомный template tag.

## Функционал

- Меню хранится в БД.
- Редактируется через админку Django.
- Активный пункт определяется по текущему URL.
- Поддержка нескольких меню на одной странице.
- Поддержка вложенных пунктов (разворачиваются родительские и первый уровень вложенности под активным пунктом).
- Переход по URL, заданному явно или через named URL.
- На отрисовку каждого меню выполняется **ровно 1 запрос к БД**.
